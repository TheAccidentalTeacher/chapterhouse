# Business Name Availability Report — Round 2

**Purpose:** Evaluate 5 potential business names for an online homeschool store  
**Research Methods:** Direct domain fetch, web search (DuckDuckGo, Bing), USPTO Trademark Electronic Search System (TESS)

## 1. Homeschool Hotspot — homeschoolhotspot.com

### Domain Status
**UNCLEAR / POSSIBLY PARKED** — The domain responded to HTTP requests but returned no extractable content. This typically indicates a parked page, a blank holding page, or a registrar landing page. The domain is registered but does not appear to host an active business website.

### Existing Businesses & Organizations Using This Name
- **"The HomeSchool HotSpot"** — A blogspot site (homeschoolhotspotjax.blogspot.com) based in Jacksonville, FL, offering private tutoring, curriculum selection & planning, and teaching strategies.
- **"904 Homeschool Hotspot"** — Listed as a business in Glen Saint Mary, Florida on Florida Student Activities (flstudentactivities.com).
- **"Homeschool Hotspot" Facebook Page** — A Facebook profile (facebook.com/homeschoolshine) using the display name "Homeschool Hotspot," active since around 2003.
- **"Homeschool Hotspot" Library Programs** — Multiple public libraries (New Milford Public Library in CT, SDG Library) use "Homeschool Hotspot" as the name for recurring homeschool programs and events.
- **Kidtivity.com** — Lists a "Homeschool Hotspot" series on their website.

### USPTO Trademark Search
**No results found** — 0 live, 0 dead trademark filings for "Homeschool Hotspot."

### Risk Level: **MEDIUM** ⚠️
### Recommendation
While there's no federal trademark, the name "Homeschool Hotspot" is already in active use by multiple parties — a Florida-based tutoring business, a Facebook page, and several library programs. The .com domain is registered (not clearly available). This creates brand confusion risk and makes it harder to establish unique identity. **Consider an alternative name.**

- **Social Media:**
  - Facebook: ~8K followers, 4.9/5 rating (188 reviews), page at facebook.com/nextlevelhomeschool
  - YouTube: @nextlevelhomeschool444
  - Instagram: @nextlevelhomeschool
  - Pinterest: /nextlevelhomesc/
- **Third-Party Presence:** Featured on Homeschool.com, Atrium (exploreatrium.com), Modulo, Homeschool Tools, Cathy Duffy Reviews, Well-Trained Mind forums, Trustpilot (82 reviews)

### USPTO Trademark Search
**No results found** — However, the business holds strong common-law trademark rights through 9+ years of continuous commercial use in the homeschool education market.

### Risk Level: **HIGH** 🔴
### Recommendation
**Do not use this name.** Next Level Homeschool is a well-established, active competitor operating in the exact same market (online homeschool education). They have a professional website, 103+ courses, strong social media presence (8K Facebook followers, 4.9/5 rating), and listings on every major homeschool directory. Even without a federal trademark registration, they have robust common-law trademark protection. Using this name would create immediate legal risk, customer confusion, and an impossible SEO battle. **This name is completely unavailable.**

## 4. Homeschool Agent — homeschoolagent.com

### Domain Status
**UNDER CONSTRUCTION (Squarespace)** — The domain is active and hosted on Squarespace. It displays a placeholder page that reads: *"homeschoolagent.com — We're under construction. Please check back for an update soon."* Someone has claimed this domain and is preparing to build on it.

### Existing Businesses & Organizations Using This Name
- **"My Homeschool Agent"** (myhomeschoolagent.com) — An existing website built on GoDaddy with the tagline "Empowering Homeschooling Families." Has a contact form and email list signup. Copyright 2025. Appears to be a new/early-stage homeschool consulting or resource business.
- **Etsy Canada** — Has marketplace listings for "Homeschool Agent Logo" designs, suggesting the concept has traction in the homeschool community.
- **Facebook** — The phrase "homeschool agent" has been used in homeschool community videos/discussions as a concept for someone who guides families through the homeschooling process.

### USPTO Trademark Search
**No results found** — 0 live, 0 dead trademark filings for "Homeschool Agent."

### Risk Level: **MEDIUM** ⚠️
### Recommendation
The .com domain is taken and under active development on Squarespace. A closely related business ("My Homeschool Agent") already exists at myhomeschoolagent.com. While there's no federal trademark, there would be brand confusion between your store and these existing entities. **Proceed with caution — the domain is unavailable and a similar brand exists.** You would need a different domain (e.g., thehomeschoolagent.com) and would compete for search visibility against both the .com holder and "My Homeschool Agent."
